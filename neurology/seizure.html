Seizure & Status Epilepticus

Stepwise bedside approach: first seizure workup, medication ladder, EEG/imaging triggers

Home Back to Neurology

Core Concepts

  • Seizure = transient episode from abnormal neuronal activity.
  • Status epilepticus (SE) = seizure that won’t stop or recurrent seizures without baseline recovery.
  • Convulsive SE is a time emergency: treat first, refine later.
  • Nonconvulsive SE can look like “delirium” — EEG is how you catch it.
Practical clock: If a convulsive seizure lasts > ~5 minutes, act like status and treat.

Stepwise Management: Convulsive Seizure / Convulsive Status

Step 0 ABCs + Immediate Reversible Causes
Do this now
  • Position patient, protect airway, suction, O2, monitor.
  • Fingerstick glucose immediately.
  • IV access; check vitals & temperature.
  • Consider thiamine if malnourished/alcohol use (before dextrose if possible).
Key labs
BMP/CMP, Mg, Phos, CBC; pregnancy test if relevant; tox if it changes management.
Localization clue
Persistent focal deficit after event → consider stroke/structural lesion.
If it’s still actively convulsing, your first goal is to stop the seizure safely.
Step 1 First-Line: Benzodiazepine
Preferred routes
  • IV lorazepam if access (common hospital choice).
  • IM midazolam if no IV access.
Practical pearl
Give an adequate dose early; under-dosing is a common reason seizures continue.
If still seizing
Repeat once (institution-dependent) while preparing second-line antiseizure medication.
Watch for: respiratory depression and hypotension, especially with repeated doses or co-ingestants.
Step 2 Second-Line: Load an Antiseizure Medication (ASM)
Option When it’s useful Key cautions
Levetiracetam (Keppra) Easy to use, minimal interactions; common first choice. Behavioral agitation can occur; dose adjust in renal dysfunction.
Valproate Good for generalized seizures; useful when avoiding hypotension. Avoid in severe liver disease, pregnancy, pancreatitis; thrombocytopenia risk.
Fosphenytoin / Phenytoin Traditional option; useful in focal seizures. Arrhythmia/hypotension with rapid infusion; many drug interactions.
Lacosamide Helpful for focal seizures; increasingly used. PR prolongation; caution in conduction disease.
Pick one second-line agent and load it promptly after benzo; don’t “wait and see” for too long if the seizure is not clearly stopping.
Step 3 Refractory Status: ICU + Continuous Infusion
Definition
Ongoing seizures despite benzo + appropriate second-line ASM.
What happens next
  • ICU-level care; intubation often needed.
  • Continuous EEG.
  • Continuous anesthetic infusion per protocol (e.g., midazolam/propofol/others depending on institution).
Goal
Seizure suppression on EEG and prevention of recurrence while treating the underlying cause.
Big picture: refractory status is often more dangerous from hypoxia, acidosis, hyperthermia, and aspiration than from the original trigger.

First Seizure (After Stabilization)

A Decide Provoked vs Unprovoked
Provoked (acute symptomatic)
  • Metabolic: hypoNa, hypoCa, hypoMg, hypo/hyperglycemia, uremia, hepatic failure
  • Toxic/withdrawal: alcohol/benzos, stimulants, meds lowering threshold
  • Structural: stroke/ICH, tumor, trauma
  • Infection: meningitis/encephalitis
Unprovoked
No acute trigger found → higher recurrence risk; consider epilepsy workup and long-term plan.
Provoked seizures are treated by fixing the trigger; long-term ASMs are individualized.
B Imaging: Who Needs CT Now vs MRI Later?
CT head now if
  • New focal deficit, persistent AMS, head trauma, anticoagulation, immunocompromised
  • Suspected hemorrhage, mass, or stroke
MRI brain (preferred) if
First unprovoked seizure, focal semiology, or CT unrevealing with persistent concern.
C EEG: When It’s High Yield
Get EEG if
  • Unexplained persistent AMS (rule out NCSE)
  • First unprovoked seizure (risk stratification)
  • Focal features or concern for subclinical seizures
What it answers
Epileptiform activity, seizure burden, encephalopathy pattern.

Nonconvulsive Status Epilepticus (NCSE)

Think of NCSE when the patient is “not waking up right” or is persistently confused without a clear reason.

Clues When to suspect NCSE
  • Prolonged post-ictal state
  • Fluctuating mental status with subtle twitching, eye deviation, automatisms
  • Unexplained coma in ICU
  • Failure to improve after correcting metabolic issues
NCSE is an EEG diagnosis. If you’re worried, ask for EEG early.

Common Seizure Mimics (Fast Differentiation)

Mimics What looks like a seizure but isn’t
  • Syncope: prodrome, pallor, quick recovery; can have brief myoclonic jerks
  • Psychogenic nonepileptic events (PNES): prolonged, asynchronous movements, eye closure, preserved awareness patterns
  • Stroke/TIA: focal deficits without the evolving motor activity pattern
  • Metabolic encephalopathy: tremor/asterixis, diffuse confusion
History and recovery pattern often beat “what it looked like” descriptions.

When to Call Neurology Practical triggers

  • Status epilepticus (convulsive or suspected nonconvulsive)
  • First unprovoked seizure with concerning features
  • Persistent focal deficits, unclear diagnosis, or abnormal imaging
  • Need for EEG interpretation or continuous EEG
